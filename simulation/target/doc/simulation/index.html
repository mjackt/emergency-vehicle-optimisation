Â§What is it?

A program to optimise the locations of police vehicles.

The program will take input files, which allow you to customise the area and police stations used for the experiment. It will then generate and improve solutions using a GA paired with a response simulation, which will provide an average response time to be used as a fitness function. An incident schedule is generated at the start of the program and every solution is tested on the same schedule to ensure fairness. This means one single run of this program will not provide a solution best to all circumstances.

Â§Tuning GA

A variety of paremeters are available to tune the GA. Some relate to the GA process, i.e tournament size or mutation count. Others refer to the simulation, i.e timestep or the max number of cars available.

By default they are set to the parameters I have found most succesfull. However, this will change based on the area and constraints being used.
